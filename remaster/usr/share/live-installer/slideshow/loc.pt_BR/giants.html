Situada sobre os ombros de gigantes

  • Remaster é uma distribuição GNU / Linux baseada no Debian e derivadas.
  • As nossas inovações, o rápido desenvolvimento do Debian e a quantidade enorme de pacotes Debian tornam o Remaster um dos sistemas operacionais mais atraentes para desktop disponível para usuários domésticos.
  • Nas casas, o Remaster tem grande potêncial para ser um sistema operacional muito usado.
